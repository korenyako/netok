# Netok SoT — Главный экран (v1.4, Актуально)

## Шапка
- RU: «Интернет работает», «IP: … (Город, Страна)», «Соединение без VPN»
- EN: "Internet available", "IP: … (City, Country)", "Connection without VPN"

## Узлы
- RU:
  - Компьютер (Имя ПК): «Сетевой адаптер: …», «IP в локальной сети: …»
  - Сеть (Тип + SSID): «Сигнал: отличный/хороший/средний/слабый (-dBm)»
  - Роутер (Модель/бренд если известны): «IP в локальной сети: …»
  - Интернет (Оператор): «Скорость: X/Y Мбит/с»
- EN:
  - Computer (Host name): "Network adapter: …", "Local IP: …"
  - Network (Type + SSID): "Signal: excellent/good/average/weak (-dBm)"
  - Router/Access Point (Model/brand if known): "Local IP: …"
  - Internet (ISP name): "Speed: X/Y Mbps"

## Нижний блок
- RU: «[Обновить] [Настройки]», затем «Обновлено HH:MM:SS»
- EN: "[Refresh] [Settings]", then "Updated HH:MM:SS"

## Правила оформления
- В заголовках узлов сразу указывать имя/название в скобках.
- В шапке не показывать скорость.
- Не использовать слова «ОК», «статус» — только факты; статус узлов — цветом.
- Timestamp выводится один раз, внизу под кнопками.

- v1.3 и старше — см. `docs/deprecated/`.
