Netok — кроссплатформенная утилита сетевой диагностики.  
Цель: показать пользователю понятную «цепочку» от компьютера до интернета.

## Главный экран

- Вертикальный путь из 4 узлов:
  1. Компьютер
  2. Сеть (Wi-Fi / Кабель / Модем)
  3. Роутер / Точка доступа
  4. Интернет

- Каждый узел показывает факты (имя устройства, IP, адаптер и т.п.).
- Состояние узла обозначается цветом бусины:
  - зелёная = доступен,
  - красная = недоступен,
  - серая = неизвестно/нет данных.

## Верхний блок

- Строка 1: Интернет работает / недоступен / частично
- Строка 2: Скорость X/Y Мбит/с
- Строка 3: Соединение без VPN (VPN в MVP не реализован — убран)
